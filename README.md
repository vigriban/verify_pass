# Password Strength Verificator 

Verifies password strength interactively

### Prerequisites
```
python3.6+
```

### Installing and running
Clone repository, install dependencies from requirements.txt and run with
```
python3 verify_pass.py
```
## Running the tests

```
python3 tests.py
```
